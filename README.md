## CVT (ELM327) — Mitsubishi Lancer X

Простое Android приложение на **Jetpack Compose**, подключается к **ELM327 по Bluetooth Classic (SPP)** и показывает текущую температуру вариатора **CVT** для Mitsubishi Lancer X.

### Что читает

- **CVT temp**: `ModeAndPID = 2103`, `Header = 7E1`
  - Формулы (как в Torque):
    - `Temp 1`: \((0.000000002344*(N^5))+(-0.000001387*(N^4))+(0.0003193*(N^3))+(-0.03501*(N^2))+(2.302*N)+(-36.6)\)
    - `Temp 2`: \((0.0000286*N*N*N)+(-0.00951*N*N)+(1.46*N)+(-30.1)\)
- **CVT oil degradation** (опционально): `ModeAndPID = 2110`, `Header = 7E1`
  - Формула: `AC*256+AD`

Переменные формул:
- `AA`, `AB`, `AC`, `AD`… — байты данных после `61 xx`
- `A`, `B`, `C`, `D`… — алиасы к `AA`, `AB`, `AC`, `AD`
- `N` — алиас к `AA` (как в вашем CSV)

### Как пользоваться

1. **Спарьте** ELM327 в системных настройках Android (Bluetooth).
2. Запустите приложение.
3. На Android 12+ выдайте разрешение **BLUETOOTH_CONNECT**.
4. Выберите устройство из списка **спаренных** и нажмите **Connect**.
5. Температура обновляется примерно **1 раз в секунду**.

### Troubleshooting

- **Нет устройства в списке**: оно не спарено в настройках Android.
- **NO DATA**: ECU не отвечает на `2103`/`2110` (не тот протокол/ECU, плохая связь, либо авто/адаптер не поддерживает запрос).
- **Connect error**: часто это неудачное спаривание/неправильный PIN/адаптер занят другим приложением.
